Alfred Gyazo Workflow

AlfredでGyazoにアップロードした画像リンクを貼り付けるワークフロー

View on GitHub Download .zip Download .tar.gz




🗻 素早くGyazoにアップロードした画像リンクを貼り付ける

以下のフォーマットで貼り付けられます。

  • Markdown
  • HTML
  • そのまま

選択した画像リンクはクリップボードにコピーされ、選択中のウィンドウがある場合は自動でペーストされます。




📦 インストール

こちらからワークフローパッケージをダウンロードして開くと、次のような確認画面が出るのでImportを押してください。




🔧 初期設定

Gyazoでアプリケーションを登録しトークンを生成する

Gyazoで[New Application]を選択してアプリケーションを登録します。

[Submit]を押して作成したアプリケーションのページで[Generate]を押してトークンを生成し、コピーしておいてください。

Alfred Gyazo Workflowにトークンを保存する

トークンは~/.gyazo/config.jsonに保存されます。
